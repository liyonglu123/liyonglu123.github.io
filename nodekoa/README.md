### node + koa +mongodb 实现简易的后台管理系统
```
    1. npm init 
    2. koa
    3. nodemon 热加载
    4. koa-router 处理app.use 只有在next()之后顺序执行的问题
    5. 原生node不支持 import的语法需要 babel-plugin-transform-es2015-modules-commonjs babel-register
    6. 结合ejs和art-template实现页面的展示 cnpm install koa-views --save  cnpm install ejs  --save

```
