# 实现一个 TicTacToe 三子棋游戏

## 游戏规则

- 棋盘：3\*3 方格
- 双方分别持有圆圈和叉两种棋子
- 双方交替落子
- 率先连成三子直线的一方获胜

- 第一层策略：我要赢
- 第二层策略： 别输
